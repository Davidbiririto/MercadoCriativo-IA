Uma plataforma para transformar produto em anúncio completo

O MarketCriativo IA foi pensado para quem vende online e precisa criar anúncios melhores sem perder tempo com texto, estrutura, palavras-chave e criativos.

📦

1. Envie o produto

Adicione foto, nome, categoria, marca, variações, público-alvo e diferenciais do produto.

🧠

2. A IA organiza tudo

O sistema cria textos e ideias respeitando o formato do marketplace escolhido: Shopee, Mercado Livre ou Amazon.

📈

3. Copie, baixe e publique

Receba título, descrição, bullets, CTA, palavras-chave, preço psicológico e criativos visuais.

Perguntas importantes

O que esta versão faz agora e o que precisa ser conectado para virar SaaS real.

O login é real?

Nesta versão MVP, o login é simulado com localStorage. Para produção, use Supabase, Firebase ou Auth.js.

O pagamento é real?

Os planos são simulados. Para cobrar assinatura, conecte Mercado Pago, Stripe, Kiwify ou Kirvano.

A IA é real?

Sim, quando a variável OPENAI_API_KEY estiver configurada no Vercel. Sem ela, o sistema cai para modo demonstração.

Gera imagem do produto?

A função de imagem envia a foto do produto para a API e pede um criativo profissional. O resultado depende da API configurada.
